## Descrizione
Questo progetto è un framework realizzato ed impiegato per simulare varie tipologie d'errore nelle misure di Spettroscopia Elettrochimica d'Impedenza, degradando la qualità delle acquisizioni in svariati modi, a partire da un dataset open [1].

## Descrizione
Questo progetto è una suite di strumenti impiegati per effettuare varie modalità di simulazione d'errore nelle misure di Spettroscopia Elettrochimica d'Impedenza, a partire da un dataset open [1].

## References

[1] M. Rashid et al., "Dataset for rapid state of health estimation of lithium batteries using EIS and machine learning: Training and validation," Data in Brief, vol. 48, p. 109157, 2023. doi:10.1016/j.dib.2023.109157
